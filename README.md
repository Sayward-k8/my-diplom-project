## Р—Р°РҙР°СҮР°
РҡР»СҺСҮРөРІР°СҸ Р·Р°РҙР°СҮР° вҖ” СҖР°Р·СҖР°РұРҫСӮР°СӮСҢ РҫСӮРәР°Р·РҫСғСҒСӮРҫР№СҮРёРІСғСҺ РёРҪС„СҖР°СҒСӮСҖСғРәСӮСғСҖСғ РҙР»СҸ СҒР°Р№СӮР°, РІРәР»СҺСҮР°СҺСүСғСҺ РјРҫРҪРёСӮРҫСҖРёРҪРі, СҒРұРҫСҖ Р»РҫРіРҫРІ Рё СҖРөР·РөСҖРІРҪРҫРө РәРҫРҝРёСҖРҫРІР°РҪРёРө РҫСҒРҪРҫРІРҪСӢС… РҙР°РҪРҪСӢС…. РҳРҪС„СҖР°СҒСӮСҖСғРәСӮСғСҖР° РҙРҫР»Р¶РҪР° СҖР°Р·РјРөСүР°СӮСҢСҒСҸ РІ [Yandex Cloud](https://cloud.yandex.com/) Рё РҫСӮРІРөСҮР°СӮСҢ РјРёРҪРёРјР°Р»СҢРҪСӢРј СҒСӮР°РҪРҙР°СҖСӮР°Рј РұРөР·РҫРҝР°СҒРҪРҫСҒСӮРё: Р·Р°РҝСҖРөСүР°РөСӮСҒСҸ РІСӢРәР»Р°РҙСӢРІР°СӮСҢ СӮРҫРәРөРҪ РҫСӮ РҫРұР»Р°РәР° РІ git. РҳСҒРҝРҫР»СҢР·СғР№СӮРө [РёРҪСҒСӮСҖСғРәСҶРёСҺ](https://cloud.yandex.ru/docs/tutorials/infrastructure-management/terraform-quickstart#get-credentials).

**РҹРөСҖРөРҙ РҪР°СҮР°Р»РҫРј СҖР°РұРҫСӮСӢ РҪР°Рҙ РҙРёРҝР»РҫРјРҪСӢРј Р·Р°РҙР°РҪРёРөРј РёР·СғСҮРёСӮРө [РҳРҪСҒСӮСҖСғРәСҶРёСҸ РҝРҫ СҚРәРҫРҪРҫРјРёРё РҫРұР»Р°СҮРҪСӢС… СҖРөСҒСғСҖСҒРҫРІ](https://github.com/netology-code/devops-materials/blob/master/cloudwork.MD).**

## РҳРҪС„СҖР°СҒСӮСҖСғРәСӮСғСҖР°
Р”Р»СҸ СҖР°Р·РІС‘СҖСӮРәРё РёРҪС„СҖР°СҒСӮСҖСғРәСӮСғСҖСӢ РёСҒРҝРҫР»СҢР·СғР№СӮРө Terraform Рё Ansible.  

РқРө РёСҒРҝРҫР»СҢР·СғР№СӮРө РҙР»СҸ ansible inventory ip-Р°РҙСҖРөСҒР°! Р’РјРөСҒСӮРҫ СҚСӮРҫРіРҫ РёСҒРҝРҫР»СҢР·СғР№СӮРө fqdn РёРјРөРҪР° РІРёСҖСӮСғР°Р»СҢРҪСӢС… РјР°СҲРёРҪ РІ Р·РҫРҪРө ".ru-central1.internal". РҹСҖРёРјРөСҖ: example.ru-central1.internal  - РҙР»СҸ СҚСӮРҫРіРҫ РҙРҫСҒСӮР°СӮРҫСҮРҪРҫ РҝСҖРё СҒРҫР·РҙР°РҪРёРё Р’Рң СғРәР°Р·Р°СӮСҢ name=example, hostname=examle !! 

Р’Р°Р¶РҪРҫ: РёСҒРҝРҫР»СҢР·СғР№СӮРө РҝРҫ-РІРҫР·РјРҫР¶РҪРҫСҒСӮРё **РјРёРҪРёРјР°Р»СҢРҪСӢРө РәРҫРҪС„РёРіСғСҖР°СҶРёРё Р’Рң**:2 СҸРҙСҖР° 20% Intel ice lake, 2-4Р“Рұ РҝР°РјСҸСӮРё, 10hdd, РҝСҖРөСҖСӢРІР°РөРјР°СҸ. 

**РўР°Рә РәР°Рә РҝСҖРөСҖСӢРІР°РөРјР°СҸ Р’Рң РҝСҖРҫСҖР°РұРҫСӮР°РөСӮ РҪРө РұРҫР»СҢСҲРө 24СҮ, РҝРөСҖРөРҙ СҒРҙР°СҮРөР№ СҖР°РұРҫСӮСӢ РҪР° РҝСҖРҫРІРөСҖРәСғ РҙРёРҝР»РҫРјРҪРҫРјСғ СҖСғРәРҫРІРҫРҙРёСӮРөР»СҺ СҒРҙРөР»Р°Р№СӮРө РІР°СҲРё Р’Рң РҝРҫСҒСӮРҫСҸРҪРҪРҫ СҖР°РұРҫСӮР°СҺСүРёРјРё.**

РһР·РҪР°РәРҫРјСҢСӮРөСҒСҢ СҒРҫ РІСҒРөРјРё РҝСғРҪРәСӮР°РјРё РёР· СҚСӮРҫР№ СҒРөРәСҶРёРё, РҪРө РұРөСҖРёСӮРөСҒСҢ СҒСҖР°Р·Сғ РІСӢРҝРҫР»РҪСҸСӮСҢ Р·Р°РҙР°РҪРёРө, РҪРө РҙРҫСҮРёСӮР°РІ РҙРҫ РәРҫРҪСҶР°. РҹСғРҪРәСӮСӢ РІР·Р°РёРјРҫСҒРІСҸР·Р°РҪСӢ Рё РјРҫРіСғСӮ РІР»РёСҸСӮСҢ РҙСҖСғРі РҪР° РҙСҖСғРіР°.

### РЎР°Р№СӮ
РЎРҫР·РҙР°Р№СӮРө РҙРІРө Р’Рң РІ СҖР°Р·РҪСӢС… Р·РҫРҪР°С…, СғСҒСӮР°РҪРҫРІРёСӮРө РҪР° РҪРёС… СҒРөСҖРІРөСҖ nginx, РөСҒР»Рё РөРіРҫ СӮР°Рј РҪРөСӮ. РһРЎ Рё СҒРҫРҙРөСҖР¶РёРјРҫРө Р’Рң РҙРҫР»Р¶РҪРҫ РұСӢСӮСҢ РёРҙРөРҪСӮРёСҮРҪСӢРј, СҚСӮРҫ РұСғРҙСғСӮ РҪР°СҲРё РІРөРұ-СҒРөСҖРІРөСҖР°.

РҳСҒРҝРҫР»СҢР·СғР№СӮРө РҪР°РұРҫСҖ СҒСӮР°СӮРёСҮРҪСӢС… С„Р°Р№Р»РҫРІ РҙР»СҸ СҒР°Р№СӮР°. РңРҫР¶РҪРҫ РҝРөСҖРөРёСҒРҝРҫР»СҢР·РҫРІР°СӮСҢ СҒР°Р№СӮ РёР· РҙРҫРјР°СҲРҪРөРіРҫ Р·Р°РҙР°РҪРёСҸ.

Р’РёСҖСӮСғР°Р»СҢРҪСӢРө РјР°СҲРёРҪСӢ РҪРө РҙРҫР»Р¶РҪСӢ РҫРұР»Р°РҙР°СӮСҢ РІРҪРөСҲРҪРёРј Ip-Р°РҙСҖРөСҒРҫРј, СӮРө РҪР°С…РҫРҙРёСӮСҒСҸ РІРҫ РІРҪСғСӮСҖРөРҪРҪРөР№ СҒРөСӮРё. Р”РҫСҒСӮСғРҝ Рә Р’Рң РҝРҫ ssh СҮРөСҖРөР· РұР°СҒСӮРёРҫРҪ-СҒРөСҖРІРөСҖ. Р”РҫСҒСӮСғРҝ Рә web-РҝРҫСҖСӮСғ Р’Рң СҮРөСҖРөР· РұР°Р»Р°РҪСҒРёСҖРҫРІСүРёРә yandex cloud.

РқР°СҒСӮСҖРҫР№РәР° РұР°Р»Р°РҪСҒРёСҖРҫРІСүРёРәР°:

1. РЎРҫР·РҙР°Р№СӮРө [Target Group](https://cloud.yandex.com/docs/application-load-balancer/concepts/target-group), РІРәР»СҺСҮРёСӮРө РІ РҪРөС‘ РҙРІРө СҒРҫР·РҙР°РҪРҪСӢС… Р’Рң.

2. РЎРҫР·РҙР°Р№СӮРө [Backend Group](https://cloud.yandex.com/docs/application-load-balancer/concepts/backend-group), РҪР°СҒСӮСҖРҫР№СӮРө backends РҪР° target group, СҖР°РҪРөРө СҒРҫР·РҙР°РҪРҪСғСҺ. РқР°СҒСӮСҖРҫР№СӮРө healthcheck РҪР° РәРҫСҖРөРҪСҢ (/) Рё РҝРҫСҖСӮ 80, РҝСҖРҫСӮРҫРәРҫР» HTTP.

3. РЎРҫР·РҙР°Р№СӮРө [HTTP router](https://cloud.yandex.com/docs/application-load-balancer/concepts/http-router). РҹСғСӮСҢ СғРәР°Р¶РёСӮРө вҖ” /, backend group вҖ” СҒРҫР·РҙР°РҪРҪСғСҺ СҖР°РҪРөРө.

4. РЎРҫР·РҙР°Р№СӮРө [Application load balancer](https://cloud.yandex.com/en/docs/application-load-balancer/) РҙР»СҸ СҖР°СҒРҝСҖРөРҙРөР»РөРҪРёСҸ СӮСҖР°С„РёРәР° РҪР° РІРөРұ-СҒРөСҖРІРөСҖР°, СҒРҫР·РҙР°РҪРҪСӢРө СҖР°РҪРөРө. РЈРәР°Р¶РёСӮРө HTTP router, СҒРҫР·РҙР°РҪРҪСӢР№ СҖР°РҪРөРө, Р·Р°РҙР°Р№СӮРө listener СӮРёРҝ auto, РҝРҫСҖСӮ 80.

РҹСҖРҫСӮРөСҒСӮРёСҖСғР№СӮРө СҒР°Р№СӮ
`curl -v <РҝСғРұР»РёСҮРҪСӢР№ IP РұР°Р»Р°РҪСҒРөСҖР°>:80` 

### РңРҫРҪРёСӮРҫСҖРёРҪРі
РЎРҫР·РҙР°Р№СӮРө Р’Рң, СҖР°Р·РІРөСҖРҪРёСӮРө РҪР° РҪРөР№ Zabbix. РқР° РәР°Р¶РҙСғСҺ Р’Рң СғСҒСӮР°РҪРҫРІРёСӮРө Zabbix Agent, РҪР°СҒСӮСҖРҫР№СӮРө Р°РіРөРҪСӮСӢ РҪР° РҫСӮРҝСҖР°РІР»РөРҪРёРө РјРөСӮСҖРёРә РІ Zabbix. 

РқР°СҒСӮСҖРҫР№СӮРө РҙРөСҲРұРҫСҖРҙСӢ СҒ РҫСӮРҫРұСҖР°Р¶РөРҪРёРөРј РјРөСӮСҖРёРә, РјРёРҪРёРјР°Р»СҢРҪСӢР№ РҪР°РұРҫСҖ вҖ” РҝРҫ РҝСҖРёРҪСҶРёРҝСғ USE (Utilization, Saturation, Errors) РҙР»СҸ CPU, RAM, РҙРёСҒРәРё, СҒРөСӮСҢ, http Р·Р°РҝСҖРҫСҒРҫРІ Рә РІРөРұ-СҒРөСҖРІРөСҖР°Рј. Р”РҫРұР°РІСҢСӮРө РҪРөРҫРұС…РҫРҙРёРјСӢРө tresholds РҪР° СҒРҫРҫСӮРІРөСӮСҒСӮРІСғСҺСүРёРө РіСҖР°С„РёРәРё.

### РӣРҫРіРё
CРҫР·РҙР°Р№СӮРө Р’Рң, СҖР°Р·РІРөСҖРҪРёСӮРө РҪР° РҪРөР№ Elasticsearch. РЈСҒСӮР°РҪРҫРІРёСӮРө filebeat РІ Р’Рң Рә РІРөРұ-СҒРөСҖРІРөСҖР°Рј, РҪР°СҒСӮСҖРҫР№СӮРө РҪР° РҫСӮРҝСҖР°РІРәСғ access.log, error.log nginx РІ Elasticsearch.

РЎРҫР·РҙР°Р№СӮРө Р’Рң, СҖР°Р·РІРөСҖРҪРёСӮРө РҪР° РҪРөР№ Kibana, СҒРәРҫРҪС„РёРіСғСҖРёСҖСғР№СӮРө СҒРҫРөРҙРёРҪРөРҪРёРө СҒ Elasticsearch.

### РЎРөСӮСҢ
Р Р°Р·РІРөСҖРҪРёСӮРө РҫРҙРёРҪ VPC. РЎРөСҖРІРөСҖР° web, Elasticsearch РҝРҫРјРөСҒСӮРёСӮРө РІ РҝСҖРёРІР°СӮРҪСӢРө РҝРҫРҙСҒРөСӮРё. РЎРөСҖРІРөСҖР° Zabbix, Kibana, application load balancer РҫРҝСҖРөРҙРөР»РёСӮРө РІ РҝСғРұР»РёСҮРҪСғСҺ РҝРҫРҙСҒРөСӮСҢ.

РқР°СҒСӮСҖРҫР№СӮРө [Security Groups](https://cloud.yandex.com/docs/vpc/concepts/security-groups) СҒРҫРҫСӮРІРөСӮСҒСӮРІСғСҺСүРёС… СҒРөСҖРІРёСҒРҫРІ РҪР° РІС…РҫРҙСҸСүРёР№ СӮСҖР°С„РёРә СӮРҫР»СҢРәРҫ Рә РҪСғР¶РҪСӢРј РҝРҫСҖСӮР°Рј.

РқР°СҒСӮСҖРҫР№СӮРө Р’Рң СҒ РҝСғРұР»РёСҮРҪСӢРј Р°РҙСҖРөСҒРҫРј, РІ РәРҫСӮРҫСҖРҫР№ РұСғРҙРөСӮ РҫСӮРәСҖСӢСӮ СӮРҫР»СҢРәРҫ РҫРҙРёРҪ РҝРҫСҖСӮ вҖ” ssh.  РӯСӮР° РІРј РұСғРҙРөСӮ СҖРөР°Р»РёР·РҫРІСӢРІР°СӮСҢ РәРҫРҪСҶРөРҝСҶРёСҺ  [bastion host]( https://cloud.yandex.ru/docs/tutorials/routing/bastion) . РЎРёРҪРҫРҪРёРј "bastion host" - "Jump host". РҹРҫРҙРәР»СҺСҮРөРҪРёРө  ansible Рә СҒРөСҖРІРөСҖР°Рј web Рё Elasticsearch СҮРөСҖРөР· РҙР°РҪРҪСӢР№ bastion host РјРҫР¶РҪРҫ СҒРҙРөР»Р°СӮСҢ СҒ РҝРҫРјРҫСүСҢСҺ  [ProxyCommand](https://docs.ansible.com/ansible/latest/network/user_guide/network_debug_troubleshooting.html#network-delegate-to-vs-proxycommand) . Р”РҫРҝСғСҒРәР°РөСӮСҒСҸ СғСҒСӮР°РҪРҫРІРәР° Рё Р·Р°РҝСғСҒРә ansible РҪРөРҝРҫСҒСҖРөРҙСҒСӮРІРөРҪРҪРҫ РҪР° bastion host.(РӯСӮРҫСӮ РІР°СҖРёР°РҪСӮ Р»РөРіСҮРө РІ РҪР°СҒСӮСҖРҫР№РәРө)

РҳСҒС…РҫРҙСҸСүРёР№ РҙРҫСҒСӮСғРҝ РІ РёРҪСӮРөСҖРҪРөСӮ РҙР»СҸ Р’Рң РІРҪСғСӮСҖРөРҪРҪРөРіРҫ РәРҫРҪСӮСғСҖР° СҮРөСҖРөР· [NAT-СҲР»СҺР·](https://yandex.cloud/ru/docs/vpc/operations/create-nat-gateway).

### Р РөР·РөСҖРІРҪРҫРө РәРҫРҝРёСҖРҫРІР°РҪРёРө
